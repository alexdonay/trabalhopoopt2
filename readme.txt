
Descrição do Programa

O programa faz o controle de um estacionamento.

As principais funções são:
- Cadastro de funcionários, as pessoas que trabalham para o estacionamento
- Tela de Login
- Cadastro de usuários, onde é a chave primária é a placa do veiculo
- Controle de diárias dos veiculos
- Recebimento das diárias
